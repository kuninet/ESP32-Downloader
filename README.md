# ESP32/ESP8266 Downloader基板

## 概要
* ESP32/ESP8266 Downloader基板です。
* ピン配置は ESP32 Downloader kitに合わせています。
* MOS FET x2個で自動書き込み回路を装備してしています。
* USBシリアルICはCH340Kを使用しています。
